oracle局域网共享

2019年3月30日
| Oracle
| 阅读

背景介绍

oracle 是性能较好的数据库，如何让其他电脑在局域网下访问同一台机器上的数据库呢？

​ 最近需要解决的问题：同一局域网下，想让其他计算机访问本地计算机上的oracle数据库。需要更改oracle数据库的某些配置信息，来实现共享。Oracle数据库配置文件位置：oracle\product\11.2.0\dbhome_1\NETWORK\ADMIN 　　本地数据库的共享通过三步操作来完成，下面是三个步骤的操作。

第一步：更改配置文件

更改本地的 tnsnames.ora配置文件，添加本地IP地址。

// 添加一条本地IP的描述，HOST填充本机的IP地址
ORCL =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.0.104)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = orcl)
    )
  )

第二步：添加监听

更改监听listener.ora配置文件，添加本地监听。 　　主要有两种方法，第一种，打开Net Manager程序，添加监听程序。展开本地，监听程序，在编辑–新建监听程序，在右侧的面板中，输入本机的IP地址，端口号默认是1521，保存即可。 第二种方法，直接在listener.ora配置文件里更改，即复制了一份DESCRIPTION代码，HOST填充本机的IP添加上去，即添加监听程序。

//添加监听程序的配置信息
LISTENER =
  (DESCRIPTION_LIST =
	(DESCRIPTION =
      (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.0.104)(PORT = 1521))
    )
  )

第三步：重启服务

重启一下Oracle数据服务（实例和监听服务TNSListener）。局域网下，其他计算机测试一下连接是否成功。
