虚拟机配置联网

By you

2019-05-18 00:00:00 +0000 UTC

背景介绍

在本地安装上VMware软件，无法连接上网络，发现时没有配置网络，网上搜索一番，记录下来，方便以后查看。

一、虚拟机上网络配置

虚拟机版本：centos7 上网 第一步：Edit->Virtual Network Editor 默认选择NAT模式 选择Restore Defaults即可(恢复默认设置) 第二步：打开网络配置文件。root身份登录系统 控制台输入命令：

// 打开网络配置文件
ls/etc/sysconfig/network-scripts

打开文件列表，记下ifcfg-ensXXXX。本机的参数为ifcfg-ens33。 控制台输入：vi /etc/sysconfig/network-scripts/ifcfg-ensXXXX(XXXX替换为自己的数值)，就是编辑网络配置文件，重点设置BOOTPROTO=dhcp，ONBOOT=yes即可。vi命令：按i键，即insert，进入文本插入编辑模式。

//设置网络配置参数
BOOTPROTO=dhcp
ONBOOT=yes

网络参数设置 1、BOOTPROTO BOOTPROTO=static 静态IP BOOTPROTO=dhcp 动态IP BOOTPROTO=none 无（不指定） 通常情况下是dhcp或者static，通过指定方式的办法来获得地址，如果没有指定的话可能会出现问题 2、ONBOOT ONBOOT是指明在系统启动时是否激活网卡，只有在激活状态的网卡才能去连接网络，进行网络通讯

修改完之后，先按Esc键，再按:键，然后输入wq，最后按回车键方可退出vim编辑器 第三步：服务启动。进入到服务页面，VMware DHCP Service和VMware NAT Service 确保它们都已启用 第四步：终端上输入命令：

// 重启linux上的网络服务
service network restart

重启，控制台上输入查询ip地址的命令：ifconfig，就可以显示出设备获取的动态ip地址。

二、本地连接虚拟机中Linux

用xshell连接工具，连接配置好的Linux虚拟机设备。 第一步：打开xshell连接设备，输入连接的ip地址。 第二步：输入账号密码，即可连接到指定的服务器设备。
